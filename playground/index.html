About HD-XAPI

HD-XAPI (High-Definition Experience API) is a semantic web extension of the xAPI specification.

OWL Ontology

Formal semantic definitions for all xAPI concepts including Statement, Actor, Verb, Activity, Result, and Context.

SHACL Validation

Machine-readable constraints that validate statements against profile requirements with detailed error reporting.

SHACL-AF Rules

Inference rules that generate PROV links, registration joins, and competency assertions automatically.

ODRL Policies

Data governance policies for anonymization, access control, and compliance at export boundaries.

VC/DID Integration

Verifiable credentials for portable achievements, role assertions, and tool authentication.

TLA Profile

Total Learning Architecture support for competency-based learning and career path tracking.
